Unillax

Unillax is a JavaScript library for websites that adds universal parallax effects to any element. Unillax is able to handle background and foreground effects with any standard HTML elements, making it universal and easy to implament for many projets.

What is Unillax?

Unillax is a JavaScript library for websites that adds universal parallax effects to any element. Unillax is able to handle background and foreground effects with any standard HTML elements, making it universal and easy to implament for a variety projets.

Features

All of the things Unillax can do

Overlay text

This is some overlay text. It will move inside this container.

Containers

Unillax have a versitile system that allows you to put any elements, including text, as parralax elements inside of a container. Additionally, the container can include a background image, which is also affected by the parralax effect.

<div class="column-2">
    <div class="unillax-container">
        <img src="/images/mountain-and-lake.jpg" class="unillax-background">
        <div class="unillax-overlay">
            <h2>Overlay text</h2>
            <p>This is some overlay text. It will move inside this container.</p>
        </div>
    </div>
</div>

Foregorund

Additionally, elements do not have to be restricted to a container. Foreground elements float freely above other elements, allowing a greater sense of depth.

<img src="/images/green.svg" class="unillax-element" data-unillax-depth="3" data-unillax-position="middle">
<img src="/images/blue.svg" class="unillax-element" data-unillax-depth="60" data-unillax-position="middle">
<img src="/images/orange.svg" class="unillax-element" data-unillax-depth="-8" data-unillax-position="middle">
<img src="/images/red.svg" class="unillax-element" data-unillax-depth="5" data-unillax-position="middle">
<img src="/images/purple.svg" class="unillax-element" data-unillax-depth="2" data-unillax-position="middle">
